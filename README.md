# Trimco Solutions Assistant

A lightweight AI assistant designed to turn Trimco’s public knowledge into a usable sales and solution support tool.

This prototype demonstrates how static website content can be transformed into a usable, queryable knowledge tool using a RAG approach.

---

## Purpose

Instead of browsing multiple pages across https://www.trimco-group.com/, this assistant allows users to:

- Query products and solutions directly  
- Retrieve relevant customer cases  
- Generate sales talking points grounded in real examples  

The goal is not to showcase AI for its own sake, but to demonstrate how AI can improve access to business knowledge and support real workflows.

---

## Positioning

Framed as a Sales / Solutions Assistant, this tool supports:

- Sales teams preparing for client conversations  
- Internal users exploring solution capabilities  
- Faster access to customer case references  

---

## Approach

- Extracted and structured content from Trimco’s website  
- Built a Retrieval-Augmented Generation (RAG) pipeline  
- Hosted knowledge base, embeddings, and agent on DigitalOcean  
- Exposed via a simple chat interface for internal use  

---

## Assistant Behavior Design

To better support real usage, the assistant was configured with a few key behaviors:

- Source transparency  
  Responses include relevant source URLs to allow users to verify and explore further  

- Bilingual response handling  
  The assistant adapts to the user’s language (Traditional Chinese / English), supporting a mixed-language working environment  

- Business-focused responses  
  Answers are guided to focus on solutions, use cases, and practical applications rather than generic descriptions  

---

## What this demonstrates

- A practical approach to turning unstructured content into usable knowledge  
- How AI can be applied to support real business workflows  
- Rapid prototyping of domain-specific assistants using RAG
